# Badda-Marine-monitoring-AI-system.
Badda is an AI powered project that focuses on monitoring unusual marine activity and demonstrates how an AI web based system can be used to analyze real life problems.

Somalia has the longest coastline on mainland Africa, stretching over 2,000 miles. Naturally, fishing is one of the most common livelihoods for Somalis residing near the coast. However, it has been reported that Somalia loses an estimated 300 million USD annually due to unreported fishing by foreign ships in its waters. For a developing country, this severly impacts the livelihoods of thousands of fishers who struggle to put food on their table.

As a Somali student,I aim to help spread awareness about illegal fishing and oil spil risks near our coastline through an AI powered marine monitorning system.

The goal of this project is to simulate an AI powered system that helps visualizes suspicious activities along the coastline,and gradually improve it project with realistic datas as I develop my abilities and understanding.
A markdown of how this would work:
1. using structered data, the system would simulate marine activity.
2. Unusual activity would be detected as either illegal fishing or oil spilling based on patterns backed by AI models
3. assign risk levels (minor risk, moderate risk, major risk)
4. present alerts based on user's understanding level.

What I used to bring this project to life is; HTML,CSS,Java script and API

Key features of my projects are:
simulated marine monitoring dashboard
detections of illegal fishing 
pollution event simulation
risk classification system

As I grow my skills, I wish to improve;  the intergration of marine data, risk analysis systems, AI detection using machine learning and providing a real descriptive map of Somalia's coastline.



In short, Version 1 would display: Marine dashboards, simulated alerts, basic visual demonstratios of coastline and beginner text mode. As i grow stronger in my skills, version 2 would include: a btter map system, data structure upgrades, scenario simulaton and AI based detection.
